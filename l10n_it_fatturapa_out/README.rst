===========================================
Italian Localization - FatturaPA - Emission
===========================================

This module allows you to generate the fatturaPA XML file version 1.3
http://www.fatturapa.gov.it/export/fatturazione/it/normativa/f-2.htm
to be sent to the Exchange System
http://www.fatturapa.gov.it/export/fatturazione/it/sdi.htm


Configuration
=============

See l10n_it_fatturapa


Usage
=====

 * Fill invoice data you need to export. For instance, in 'related documents' TAB,
    or in 'related documents' section within invoice line.
 * Select N invoices and run 'Export FatturaPA' wizard
